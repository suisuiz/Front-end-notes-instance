# `Uni-app` 知识点

## 强制横屏、竖屏

```javascript
plus.screen.lockOrientation('portrait-primary');竖屏

plus.screen.lockOrientation('landscape-primary');横屏
```

## 获取状态栏高度

```javascript
uni.getSystemInfoSync().statusBarHeight
```
